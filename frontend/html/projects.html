整合功能區

即時聊天室

使用 WebSocket 技術實作，用戶登入後可即時訊息交流，具備身份驗證、歷史訊息查詢與系統提示訊息功能。

個人記帳系統

透過 RESTful API 與 PostgreSQL 建立金流管理機制，支援每日支出記錄、Excel 匯出/上傳資料、資料表呈現與使用者身份綁定。

AI 智能對話（RAG 模型）

使用 Retrieval-Augmented Generation 架構整合 OpenAI API，規劃用於自動回覆技術能力與專案相關問答，目前功能開發中。

開始互動

股票追蹤系統

整合 WebSocket、即時網頁爬蟲與自製股票損益計算邏輯，可追蹤即時價格、計算獲利損益、顯示趨勢圖與匯出 Excel，介面支援使用者輸入與資料儲存。

查看股票追蹤

飲食紀錄系統

使用 GraphQL API 記錄每日飲食項目，包括熱量、蛋白質、脂肪、碳水與份量等健康資訊，支援欄位自訂、資料查詢與 Excel 匯出，方便進行長期飲食控管與數據分析。
