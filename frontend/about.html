關於我

您好！我是王偉任，來自基隆市，是一名應屆畢業生，畢業於中原大學應用數學研究所，主修資訊組。除了學業，我熱愛桌球，曾在大學及研究所期間參加校隊，並於大二及大四擔任副隊長與隊長。此外，我曾應聘為國小校隊助理教練，這段經歷讓我將興趣與工作結合，也讓我學到了寶貴的經驗。

球隊的經驗讓我在團隊合作與人際溝通方面大有提升，也讓我在活動籌備與管理事務上更加熟練。同時，長期的訓練與比賽培養了我在壓力下保持冷靜與應變的能力，這些經驗對我的職業發展帶來了深遠的影響。

求學歷程

我的程式設計能力主要來自於大學與研究所的課程與研究。在學習初期，我使用 C 與 C++ 建立程式基礎，學習資料結構與物件導向概念。之後，我進一步使用 Python 作為主要的開發工具，應用於 Web 開發、GUI 介面設計與資料處理。

論文研究

我的研究專注於 Web 應用程式的開發，設計了一套教學輔助系統，透過資料庫記錄學生使用情況，並撰寫自動化程式進行數據整理，最後使用 R 進行統計檢定，以評估應用程式對學生學習成效的影響。

技術方面，該系統採用 MVC 架構，後端使用 Python Flask，前端則運用 HTML、CSS 和 JavaScript，並部署於 Heroku，使用其提供的 PostgreSQL 資料庫。

關於免役

關於我的免役狀況，我希望在此做補充說明。我的健康狀況良好，並無疾病，但因為體重超標而被判定免役。當時身高 177 公分，體重達 120 公斤，未達兵役標準。然而，在臨近畢業的半年內，我開始積極減重，目前已降至 85 公斤，並持續維持健康的生活方式。

學習規劃

我的學習重心在 Web 開發，並希望進一步補足相關技術。目前，我熟悉 Python Flask 與 Heroku，接下來希望學習前端框架如 React、Vue.js、Angular，以及後端技術如 Django、Spring Boot。同時，我也計畫探索主流雲端運算平台，提升系統部署與維運能力。

此外，在學期間我曾修過機器學習相關課程，對其有基礎認識，未來希望有機會進一步研究，嘗試參數調整與模型訓練，拓展自己的技術能力。
