關於我

您好！我是王偉任，來自基隆市，畢業於中原大學應用數學研究所，主修資訊組。除了學業，我熱愛桌球，曾在大學及研究所期間參加校隊，並於大二及大四擔任副隊長與隊長。此外，我曾應聘為國小校隊助理教練，這段經歷讓我將興趣與工作結合，也讓我學到了寶貴的經驗。

求學與技術發展

我的程式設計能力主要來自於大學與研究所的課程與研究。在學習初期，我使用 C 與 C++ 建立程式基礎，學習資料結構與物件導向概念。之後，我進一步使用 Python 作為主要的開發工具，應用於 Web 開發、GUI 介面設計與資料處理。此外，我的研究專注於 Web 應用程式的開發，設計了一套教學輔助系統，透過資料庫記錄學生使用情況，並撰寫自動化程式進行數據整理，最後使用 R 進行統計檢定，以評估應用程式對學生學習成效的影響。該系統後端採用 Python Flask，前端則運用 HTML、CSS 和 JavaScript，並部署於 Heroku，使用其提供的 PostgreSQL 資料庫。

工作經驗

畢業後，我在聯發科技擔任 Web 後端工程師，專注於 Golang Gin 框架開發，並參與 CI/CD、自動化部署與系統維運工作。我的主要工作內容包括：

  • 使用 Golang Gin 開發與維護 RESTful API 及 gRPC 服務
  • 撰寫後端功能 API 及 Unit Test，確保系統穩定性
  • 參與 CI/CD 工作流程，修改並優化 Gitea Action YAML 部署流程
  • 維運 TPPA+ 數據分析系統，處理 Excel 檔案的上傳、下載與數據分析

自主學習與個人專案

除了工作內容，我持續精進技術，並利用 AWS 搭建個人專案：

  • 學習並運用 AWS EC2、AWS Route 53、Nginx 進行網站部署
  • 使用 GitHub Actions 進行自動化部署，確保系統穩定上線
  • 配置 Nginx 反向代理，提高網站效能與安全性
